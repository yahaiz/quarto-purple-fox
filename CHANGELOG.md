# Changelog

All notable changes to the Quarto Purple Fox extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-18

### Added
- Initial release of Quarto Purple Fox format extension
- Light and dark mode support with theme toggle
- Obsidian-style callout support via Lua filter
- Support for callout types: tip, note, warning, caution, important
- Enhanced styling for:
  - Callouts with rounded corners and shadows
  - Colored borders for different callout types
- Comprehensive example document showcasing all features
- Template file for quick start
- Complete documentation in README.md
- MIT License
- .gitignore for clean repository
